# Discord Busy Bot

A simple Discord bot that replies with "Hello, I am busy" after 15 seconds when pinged.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Get your bot token from [Discord Developer Portal](https://discord.com/developers/applications)

3. Replace `YOUR_DISCORD_BOT_TOKEN_HERE` in `bot.py` with your actual token

4. Run the bot:
   ```bash
   python bot.py
   ```

## Usage

Simply mention the bot in Discord (`@BotName`) and it will reply with "Hello, I am busy" after 15 seconds.
